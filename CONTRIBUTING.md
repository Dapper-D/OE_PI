# Contributing to GhostHID

We’re glad you’re here! GhostHID is a community-driven project. Whether you are a mathematician, a kernel hacker, or a beta tester, your help is welcome.

## 🛠️ How to Contribute
1. **Fork the Repo:** Create your own copy of the project.
2. **Create a Branch:** Name it something descriptive (e.g., `feature-better-bezier`).
3. **Commit Changes:** Keep your code clean and commented.
4. **Submit a Pull Request:** Explain exactly what your change does and why it helps.

## 🧪 Areas We Need Help With
* **Movement Logic:** Refining the Bezier curves to be even more "human."
* **USB Descriptors:** Adding more profiles (Dell, HP, Microsoft keyboards).
* **Beta Testing:** Reporting how GhostHID performs on different Citrix/VMware versions.

## 💬 Communication
Please open an **Issue** before starting major work so we can coordinate efforts!
